jsFiddle Files

First Fiddle

Este es un ejemplo de un pequeño programa donde, escribiendo tu nombre y pulsando aceptar,
aparecerá un mensaje donde el programa te saluda y te dice la cantidad de clicks que has hecho
en el boton "Aceptar".



Simple Structure

Este pequeño programa es muy similar al anterior pero con una pequeña diferencia. En este programa
escribes tu nombre, le das a aceptar y saltará una alerta de bienvenida; además del mensaje abajo
del programa anterior. Pero, si no escribes tu nombre, saltará una alerta de que has de escribir tu
nombre.



Complex Structure

Este programa es un selector de aviones. Hay tres aviones para elegir y una vez que lo seleccionas
Te envia un mensaje diciendo que disfrutes del avión y te hace un breve resumen sobre la información
del avión seleccionado.



innerHTML & DocumentWrite

Es un programa muy simple donde se muestra que es lo que hace el innerHTML y el Document.Write.
Consiste en que cuando aparezca algo en el HTML has de pulsar el boton, la gracia es que jamás aparecerá
nada entonces nunca debería de aparecer nada y, si lo hiciese, aparecería un prompt diciendote "Hello there".



Window Alert's

Es un programa muy simple para descubrir las funcionalidades del alert, prompt y confirm de JavaScript.
Cuando clickas encima de cada boton aparece la funcionalidad de este indicandote que lo que ves es el
funcionamiento de este.



Console.Log

Es un programa donde genera un único cubo con unas características determinadas. Es IMPORTANTE
saber que para ver un resultado en este fiddle es necesario abrir la consola de este.



Validate

Es un programa donde llevamos a prueba la forma de validar los datos introducidos por el usuario
a través de diferentes "constraints".



Old Windows Calculator

Es una pequeña calculadora que sirve para hacer operaciones simples. Está hecha a base de condicionales.



Callbacks

Es un pequeño ejemplo de como utilizar los "callbacks" y la funcion "setInterval".
